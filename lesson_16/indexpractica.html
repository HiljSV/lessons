Приклад 2

В form атрибут method - метод передачі даних GET (за замовченням) - безпосередньо в адресному рядку POST - приховано
Формат передачі данних з HTML-форми: (так звана пара - ім'я та значення) name=value
На сервері обробляються імена та їх значення

Приклад з Google: Шукаємо наприклад "can i use" Отримуємо адресний рядок: https://www.google.com/search?q=can+i+use&...інші дані...
Дані передались методом GET
https://www.google.com/search - значення атрибуту action форми ? - відокремлення адреси від даних q - ім'я поля куди ми вводили пошуковий запит can+i+use - це данні які ми ввели
Передається пара - q=can+i+use

HTML code

form method="GET" class="form" action="#"

form method="POST" class="form" action="#"

Приклад 3

В form атрибут enctype - кодування даних під час відправки на сервер.

Для передачі файлів використовуємо - multipart/form-data Детальніше: https://www.w3schools.com/tags/att_form_enctype.asp

HTML code

form enctype="multipart/form-data" method="GET" class="form" action="#"

form enctype="multipart/form-data" method="POST" class="form" action="#"

Теги

Тег form - блочний

Тег label - рядковий

Теги input, textarea, button - блочно-рядкові

Тег input

input - Багатофункціональний елемент вводу даних. задопомогою якого передаються данні.

Атрибути

type - тип елементу вводу даних ( Усі типи https://www.w3schools.com/html/html_form_input_types.asp )

name - ім'я поля (даних)

value - значення поля за замовченням (дані)

placeholder - підпис поля

autocomplete - дозволяє (за замовченням) / забороняє (значення off) автозаповнення поля

required - обо'язково для заповнення

tabindex - керує навігацією за допомогою TAB

maxlength - максимальна кількість символів доступних для вводу

minlength - вимога до мінімальної кількості символів. працює в парі з required

checked (для деяких типів) - вибраний за замовченням ( не працює з type="text в input " )

disabled - поле перестає бути доступним для користування, данні передані не будуть

type - тип елементу вводу даних

Приклад 27 button - Кнопка

type="button" - кнопка нічого не робить

type="reset" - обнуляє (скидає) поля вводу у формі. Має знаходитить в середені тегу form.

type="submit" - відправляє форму. Має знаходитить в середені тегу form.

HTML code

button type="button" Я просто кнока button button type="reset" Я кнока яка чистить форму button button type="submit" Я кнопка яка відправляє форму button
